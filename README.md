# Movie DB

A Javascript/HTML project implementing SQL functions

## Dependencies

[Node.js / npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm): Allows Javascript to execute outside of a web browser / Package manager for Node.js

[Sqlite3](https://www.npmjs.com/package/sqlite): SQL library

<ins>In the Terminal</ins>
*Git clone https://github.com/AFay-OU/MovieDB.git*

<ins>Open the Downloaded Folder and in the Terminal</ins>
*npm install*
*node server.js*
